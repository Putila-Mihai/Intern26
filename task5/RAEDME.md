Incercati sa identificati ce interfata functionala ar trebui folosita pentru urmatoarele puncte si rezolvati-le folosind appropriate lambdas sau method references, unde este cazul.
1. Write a lambda expression that takes two integers and returns their sum.
2. Create a method that tests whether a string is not null and not empty.
3. Use a lambda expression with Collections.sort() to sort a list of integers in descending order.
4. Create a method that removes spaces from a given string and converts it to uppercase
5. Write a method that takes a Predicate<Integer> and an Integer as arguments. The method should print "Accepted" if the predicate returns true for the integer, otherwise print "Rejected".
6. Having a list of Person objects, create a method that modifies the names of the persons by appending " Mr./Ms." before it.
7. Create a method that returns a random value and display the result
8. Write a function that calculates the product of two integers and adds a constant value (e.g., 10) to the result.
9. Define your own functional interface. Choose what it should do.
